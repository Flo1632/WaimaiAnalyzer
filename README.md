# рҹҘЎ WaimaiAnalyzer

**WaimaiAnalyzer** is a machine-learning-powered sentiment analysis tool for Chinese food delivery (еӨ–еҚ–, *waimai*) reviews. Enter a review in **any language** вҖ” it gets translated to Chinese automatically вҖ” and the app instantly tells you whether the sentiment is **positive** рҹҳӢ or **negative** рҹҳЎ, complete with a confidence percentage.

---

## вңЁ Features

- рҹҢҸ **Multi-language support** вҖ” Accepts reviews in any language and auto-translates them to Chinese (Simplified) via Google Translate before analysis.
- вңӮпёҸ **Chinese text segmentation** вҖ” Uses [Jieba](https://github.com/fxsjy/jieba) to tokenise Chinese text the same way the model was trained.
- рҹӨ– **Naive Bayes classifier** вҖ” A lightweight, fast `CountVectorizer + MultinomialNB` pipeline trained on ~10,000 real-world Waimai reviews.
- рҹ“Ҡ **Confidence scores** вҖ” Shows the exact probability for both positive and negative predictions so you always know how certain the model is.
- рҹҺҲ **Visual feedback** вҖ” Balloons for positive reviews, clear error styling for negative ones.
- вҡЎ **Streamlit web UI** вҖ” A clean, interactive browser-based interface that anyone can use without writing a single line of code.

## рҹҡҖ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Flo1632/WaimaiAnalyzer.git
cd WaimaiAnalyzer
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Train the model *(optional вҖ” a pre-trained model is already included)*

```bash
python main.py
```

This reads `data/waimai_10k.csv`, trains the classifier, prints the accuracy, and saves the model to `model/waimai_model.pkl`.

### 4. Launch the web app

```bash
streamlit run app.py
```

Open your browser at `http://localhost:8501` and start analysing reviews!

## рҹ§  How It Works

1. **Input** вҖ” The user types a review (any language) into the text field.
2. **Translation** вҖ” If the *auto-translate* checkbox is enabled, the text is sent to Google Translate and converted to Simplified Chinese.
3. **Tokenisation** вҖ” Jieba splits the Chinese text into individual word tokens (e.g. `"еӨ–еҚ–еҫҲжЈ’"` вҶ’ `["еӨ–еҚ–", "еҫҲ", "жЈ’"]`).
4. **Vectorisation** вҖ” `CountVectorizer` converts the token list into a bag-of-words feature vector.
5. **Classification** вҖ” `MultinomialNB` predicts the sentiment label (`0` = negative, `1` = positive) and returns class probabilities.
6. **Display** вҖ” The app shows the result, confidence score, and a per-class probability breakdown.
